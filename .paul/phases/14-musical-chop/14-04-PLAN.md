<objective>
## Goal
Add a `chop_mode` ComboBox to the plugin UI (Fixed / Beat / Onset / Bar / Energy / Structural), wire it through AudioBridge to the sidecar, and inject the selected chop op into the effect chain at remix time.

## Purpose
The 5 new ops from Plan 14-02 are only reachable via preset JSON. This plan exposes chop mode as a live UI control so users can switch between musical chop algorithms without manually editing presets.

## Output
- Python: `chop_mode` field in RemixRequest + op injection in POST /api/v1/remix
- C++: `chop_mode` in RemixParams + sent in AudioBridge JSON + ComboBox in editor
</objective>
