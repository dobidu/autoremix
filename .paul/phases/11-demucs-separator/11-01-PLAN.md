<objective>
## Goal
Implement `DemucsSeparator` — a production-quality ML stem separator using Meta's
Demucs (htdemucs model). Register it alongside `AlgorithmicSeparator`. Increase the
C++ AudioBridge separation timeout from 30s to 300s to accommodate CPU inference time.
Update `requirements.txt` with the optional demucs dep.

## Purpose
The algorithmic FFT separator is the weakest link in the stack: it band-splits without
source separation, producing leaky stems. Demucs replaces it with a state-of-the-art
deep-learning model that produces clean vocals/drums/bass/other isolation.
This single change is the biggest quality improvement possible without touching any
other code.

    Create `python/server/separators/demucs_sep.py`:

    ```python
    from pathlib import Path
    from .base import IStemSeparator, StemPaths

    _model = None  # module-level cache — loaded once on first separate() call


    def _get_model():
        global _model
        if _model is None:
            from demucs.pretrained import get_model
            _model = get_model("htdemucs")
            _model.eval()
        return _model


    class DemucsSeparator(IStemSeparator):
        @property
        def separator_id(self) -> str:
            return "demucs"

        @property
        def display_name(self) -> str:
            return "Demucs (ML)"

        def is_available(self) -> bool:
            try:
                import demucs  # noqa: F401
                import torch    # noqa: F401
                import torchaudio  # noqa: F401
                return True
            except ImportError:
                return False

        def separate(self, input_path: Path, output_dir: Path) -> StemPaths:
            import torch
            import torchaudio
            from demucs.apply import apply_model

            model = _get_model()
            output_dir.mkdir(parents=True, exist_ok=True)

            wav, sr = torchaudio.load(str(input_path))

            if sr != model.samplerate:
                wav = torchaudio.functional.resample(wav, sr, model.samplerate)

            if wav.shape[0] == 1:
                wav = wav.repeat(2, 1)

            with torch.no_grad():
                sources = apply_model(model, wav.unsqueeze(0))
            # sources: [batch=1, num_sources, channels, samples]

            stem_paths = {}
            for i, name in enumerate(model.sources):
                stem = sources[0, i]  # [channels, samples]
                out = output_dir / f"{name}.wav"
                torchaudio.save(str(out), stem.cpu(), model.samplerate)
                stem_paths[name] = out

            return StemPaths(
                vocals=stem_paths["vocals"],
                drums=stem_paths["drums"],
                bass=stem_paths["bass"],
                other=stem_paths["other"],
            )
    ```

    Notes:
    - `_model` is module-level — loaded once per process, survives across requests (AC-3)
    - `is_available()` catches `ImportError` → False when demucs/torch not installed (AC-1)
    - `model.sources` gives the canonical stem order from the model metadata — no hardcoding
    - `.cpu()` ensures the stem tensor is on CPU before saving, regardless of inference device
    - All heavy imports are inside methods — import-time cost is zero when demucs absent
