<task type="auto">
  <name>Task 1: Write JSON Schema for RemixPreset</name>
  <files>python/server/presets/schema.json</files>
  <action>
    Create `python/server/presets/schema.json` with JSON Schema draft-07.

    Required fields: `id`, `version`, `name`, `engine`, `params`
    Optional fields: `description`, `author`, `tags`, `stem_mix`, `effects`

    Schema:
    ```json
    {
      "$schema": "http://json-schema.org/draft-07/schema#",
      "$id": "https://autoremix/schemas/remix-preset-1.0.json",
      "title": "RemixPreset",
      "description": "Declarative remix mode definition. engine field maps to a registered IRemixEngine. effects is reserved for Phase 10 DSL pipeline.",
      "type": "object",
      "required": ["id", "version", "name", "engine", "params"],
      "additionalProperties": false,
      "properties": {
        "id": {
          "type": "string",
          "pattern": "^[a-z][a-z0-9_]*$",
          "description": "Unique snake_case identifier. Used as engine_id in RemixRequest."
        },
        "version": {
          "type": "string",
          "pattern": "^\\d+\\.\\d+$",
          "description": "Preset schema version (major.minor)."
        },
        "name": {
          "type": "string",
          "maxLength": 32,
          "description": "Display name shown in UI tab bar."
        },
        "description": {
          "type": "string",
          "maxLength": 256
        },
        "author": {
          "type": "string",
          "maxLength": 64
        },
        "tags": {
          "type": "array",
          "items": { "type": "string" },
          "maxItems": 10
        },
        "engine": {
          "type": "string",
          "description": "Registered engine ID to use for processing (e.g. chopped_screwed)."
        },
        "params": {
          "type": "object",
          "required": [
            "tempo_factor", "pitch_shift_semi", "reverb_mix",
            "chop_interval_ms", "bass_boost_db", "drums_tempo_factor"
          ],
          "additionalProperties": false,
          "properties": {
            "tempo_factor":       { "type": "number", "minimum": 0.1, "maximum": 4.0 },
            "pitch_shift_semi":   { "type": "number", "minimum": -24.0, "maximum": 24.0 },
            "reverb_mix":         { "type": "number", "minimum": 0.0, "maximum": 1.0 },
            "chop_interval_ms":   { "type": "number", "minimum": 0.0, "maximum": 10000.0 },
            "bass_boost_db":      { "type": "number", "minimum": -24.0, "maximum": 24.0 },
            "drums_tempo_factor": { "type": "number", "minimum": 0.1, "maximum": 4.0 }
          }
        },
        "stem_mix": {
          "type": "object",
          "required": ["vocals", "drums", "bass", "other"],
          "additionalProperties": false,
          "properties": {
            "vocals": { "type": "number", "minimum": 0.0, "maximum": 2.0 },
            "drums":  { "type": "number", "minimum": 0.0, "maximum": 2.0 },
            "bass":   { "type": "number", "minimum": 0.0, "maximum": 2.0 },
            "other":  { "type": "number", "minimum": 0.0, "maximum": 2.0 }
          }
        },
        "effects": {
          "type": "array",
          "items": {},
          "description": "Reserved for Phase 10 effect-chain DSL. Must be empty in version 1.x presets."
        }
      }
    }
    ```

    Note: `stem_mix` is optional at schema level (not in `required`) because 09-02 will default
    all stems to 1.0 when absent. `effects` must be an empty array in v1.x.
  </action>
  <verify>python3 -c "import json; s=json.load(open('python/server/presets/schema.json')); print('schema fields:', list(s['properties'].keys()))"</verify>
  <done>AC-1 partially: schema file exists and is valid JSON with all expected properties</done>
</task>
