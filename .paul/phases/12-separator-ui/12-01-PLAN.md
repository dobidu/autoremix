<objective>
## Goal
Add separator selection ComboBox to the plugin UI so the user can choose between
Algorithmic FFT and Demucs (ML) separation at runtime. Fix the hardcoded
`/tmp/autoremix` Unix-only paths. Remove orphaned StyleTabBar.h. Bump to v2.4.0.

## Purpose
Phase 11 added Demucs but the plugin always passes `separator_id="algorithmic"` —
the user has no way to leverage the ML separator from the UI. The `/tmp/autoremix`
paths also break on Windows. This closes both gaps and cleans up the orphaned file.

## Output
- `AudioBridge::getAvailableSeparators()` returning `vector<string>` of IDs
- `SeparatorInfo` struct in PluginTypes.h
- `separator_combo_` ComboBox in plugin header (next to style_combo_)
- `onClick_Play` passes selected separator ID, not hardcoded `"algorithmic"`
- Temp paths use `juce::File::getSpecialLocation(tempDirectory)` — cross-platform
- StyleTabBar.h deleted
- CMakeLists.txt version 2.4.0
</objective>
