# 07-03 Plan: Waveform Display

## Goal

Render juce::AudioThumbnail waveform in plugin UI after user loads a file. Gives visual feedback on loaded audio; replaces blank space between ComboBox and Play button.

## Acceptance Criteria

| ID | Criterion |
|----|-----------|
| AC-1 | Waveform area visible (dark surface, rounded rect) after Load |
| AC-2 | Waveform fills area proportionally; mono or stereo both correct |
| AC-3 | Area shows flat/empty state before any file loaded |
| AC-4 | Build clean (0 errors, 0 warnings) |
| AC-5 | Visual sign-off from user (checkpoint:human-verify) |

## Scope

**No CMakeLists change needed** — `juce_audio_utils` already depends on `juce_audio_formats`; `AudioThumbnail`, `AudioThumbnailCache`, `AudioFormatManager` available.

Window grows from 480×220 → 480×340 (+120px for waveform).

## Tasks

### Task 1 — Members + formatManager registration (PluginEditor.h + .cpp)

**plugin/include/PluginEditor.h** — add private members:
```cpp
juce::AudioFormatManager   format_manager_;
juce::AudioThumbnailCache  thumbnail_cache_{5};
juce::AudioThumbnail       thumbnail_{512, format_manager_, thumbnail_cache_};
```

**plugin/src/AutoRemixEditor.cpp** — constructor:
```cpp
format_manager_.registerBasicFormats();  // WAV, AIFF, FLAC, MP3
```

### Task 2 — Load waveform on file pick

In `loadFile()` callback, after setting `file_path_`:
```cpp
juce::File f(file_path_);
thumbnail_.setSource(new juce::FileInputSource(f));
repaint();
```

### Task 3 — Paint waveform + update bounds

**paint()** — draw waveform into waveform rect:
```cpp
juce::Rectangle<int> waveArea(10, 90, getWidth()-20, 96);
g.setColour(juce::Colour(AR::SURFACE));
g.fillRoundedRectangle(waveArea.toFloat(), 4.0f);
if (thumbnail_.getTotalLength() > 0.0) {
    g.setColour(juce::Colour(AR::PURPLE));
    thumbnail_.drawChannels(g, waveArea, 0.0, thumbnail_.getTotalLength(), 1.0f);
} else {
    g.setColour(juce::Colour(AR::COMMENT));
    g.drawFittedText("No file loaded", waveArea, juce::Justification::centred, 1);
}
```

**drawAndConfigComponents()** — update setBounds calls:
```cpp
// window
juce::Component::setSize(480, 340);

// existing rows shift down
loadfile_btn.setBounds(10, 10, 140, 30);
file_lbl.setBounds(158, 10, 312, 30);
remix_selector_lbl.setBounds(10, 52, 60, 24);
remix_selector.setBounds(80, 50, getWidth()-90, 30);
// waveform area painted, not a component

const int btnY = 200;
play_btn.setBounds(startX,      btnY, 80, 36);
save_btn.setBounds(startX+90,   btnY, 80, 36);

status_lbl.setBounds(10, 248, getWidth()-20, 24);
progress_bar_.setBounds(10, 278, getWidth()-20, 18);

// surface separator y = 194
```

Update `paint()` separator line: `g.fillRect(0, 194, getWidth(), 1);`

- Empty state shows "No file loaded"
- Buttons/labels not overlapping
- Dark background intact
