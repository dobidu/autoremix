  <action>
    Replace the 2-line stub with a complete README. Sections:

    **# AutoRemix v2**
    One-paragraph description: JUCE VST3/Standalone plugin + Python FastAPI sidecar.
    Pluggable stem separation and remix engines. 3 remix modes: Chopped & Screwed,
    Slowed + Reverb, Drum and Bass.

    **## Requirements**
    - CMake ≥ 3.22, Ninja, C++20 compiler (GCC 12+ or Clang 14+)
    - Python 3.12, uv
    - JUCE 7 (fetched as submodule)
    - libcpr, nlohmann/json (fetched via FetchContent)

    **## Build**
    ```bash
    git clone --recurse-submodules https://github.com/dobidu/autoremix.git
    cd autoremix
    cmake -B build -G Ninja -DCMAKE_BUILD_TYPE=Debug
    cmake --build build --parallel
    # Artifacts:
    #   build/plugin/AutoRemix_artefacts/Debug/Standalone/AutoRemix
    #   build/plugin/AutoRemix_artefacts/Debug/VST3/AutoRemix.vst3/
    ```

    **## Python Sidecar**
    ```bash
    cd python
    uv venv
    uv pip install -r requirements.txt
    python server/main.py          # runs on port 17432
    # or let the plugin auto-start it:
    export AUTOREMIX_SERVER_PATH=$(pwd)/server/main.py
    ```

    **## Running the Plugin (Standalone)**
    ```bash
    # With auto-start:
    AUTOREMIX_SERVER_PATH=/path/to/autoremix/python/server/main.py \
      ./build/plugin/AutoRemix_artefacts/Debug/Standalone/AutoRemix
    # Then: Load File → select WAV → choose remix style → Play
    ```

    **## Running Tests**
    ```bash
    cd python
    .venv/bin/python -m pytest tests/ -v
    ```

    **## Architecture**
    Brief: plugin (C++/JUCE) ↔ HTTP IPC ↔ Python sidecar (FastAPI, port 17432).
    Stem separation and remix engines are pluggable via IStemSeparator / IRemixEngine.
    Registries allow new backends without modifying existing code.

    **## Remix Modes**
    Table: engine_id | description | default params

    **## License**
    GPL-3.0

    Keep it factual and concise — no marketing fluff.
