<objective>
## Goal
Extend the template schema (NativePresetTypes.h) with a `StructureConfig` block
that lets templates declare structure-aware behaviour: arrangement slots, loop
targets, scratch positions, drop anchors, and stem layer schedules per section.

## Purpose
`SongStructure` (31-01/02) is the runtime analysis. The template schema is the
static intent. This plan bridges them: a template author writes
`"drop_at": "energy_peak"` and the runtime resolves that to a concrete bar index
using the SongStructure. Without this plan, Phase 31-04 and all future
structure-aware engines have no typed input to work with.

## Output
- `NativePresetTypes.h`: `SectionLayer`, `ArrangementSlot`, `StructureConfig`
  structs; `NativeRemixPreset` gets an optional `structure` field.
- `NativePresetLoaders.h`: parse `"structure"` JSON block into `StructureConfig`.
</objective>

    // ── Structure-aware template types (Phase 31-03) ──────────────────────

    // Per-section stem layer override: which stems are active and at what gain.
    struct SectionLayer {
        float vocals = 1.0f, drums = 1.0f, bass = 1.0f, other = 1.0f;
        // Inline effect ops to apply in this section (effect_chain op ids)
        std::vector<std::string> ops;
    };

    // One slot in the arrangement sequence.
    struct ArrangementSlot {
        // Section type: "intro", "verse", "chorus", "bridge", "drop",
        // "build", "break", "outro" — or any custom string.
        std::string section_type;

        // Length in bars. 0 = auto-detect from section_boundaries.
        int length_bars = 0;

        // Source anchor: "first", "second", "last", "energy_peak",
        // "pre_chorus", "post_drop", "bar:N" (literal bar index).
        std::string source_anchor = "first";

        // Stem layer for this slot (overrides template-level stem_mix)
        SectionLayer layer;

        // Effect chain ops specific to this slot
        std::vector<std::string> effects;

        // If true, loop this slot N times before advancing
        int loop_count = 1;
    };

    // Top-level structure configuration for a preset.
    struct StructureConfig {
        bool enabled = false;

        // Arrangement sequence. If empty, engine falls back to legacy behaviour.
        std::vector<ArrangementSlot> arrangement;

        // Global scratch config (overridden per-slot if needed)
        struct ScratchConfig {
            bool        enabled    = false;
            std::string position   = "pre_drop";  // or "bar:N", "phrase:N"
            std::string style      = "baby";       // baby|transformer|chirp|forward
            int         duration_bars = 1;
            std::string stem       = "mix";        // mix|vocals|drums
        } scratch;

        // Build-up: auto-insert filter sweep + riser before drop
        struct BuildUpConfig {
            bool enabled      = false;
            int  length_bars  = 4;
            bool add_riser    = true;    // white noise riser
            bool filter_sweep = true;   // LPF opening
        } build_up;
    };
    ```

    Then in `NativeRemixPreset`, add ONE new optional field at the end:

    ```

    ```cpp
    if (j.contains("structure")) {
        const auto& s = j["structure"];
        p.structure.enabled = s.value("enabled", false);

        // Parse arrangement array
        if (s.contains("arrangement") && s["arrangement"].is_array()) {
            for (const auto& slot_j : s["arrangement"]) {
                ArrangementSlot slot;
                slot.section_type  = slot_j.value("section", "");
                slot.length_bars   = slot_j.value("length_bars", 0);
                slot.source_anchor = slot_j.value("anchor", "first");
                slot.loop_count    = slot_j.value("loop", 1);
                if (slot_j.contains("effects") && slot_j["effects"].is_array())
                    for (const auto& e : slot_j["effects"])
                        slot.effects.push_back(e.get<std::string>());
                if (slot_j.contains("layer")) {
                    const auto& ly = slot_j["layer"];
                    slot.layer.vocals = ly.value("vocals", 1.0f);
                    slot.layer.drums  = ly.value("drums",  1.0f);
                    slot.layer.bass   = ly.value("bass",   1.0f);
                    slot.layer.other  = ly.value("other",  1.0f);
                }
                p.structure.arrangement.push_back(std::move(slot));
            }
        }

        // Parse scratch block
        if (s.contains("scratch")) {
            const auto& sc = s["scratch"];
            p.structure.scratch.enabled      = sc.value("enabled", false);
            p.structure.scratch.position     = sc.value("position", "pre_drop");
            p.structure.scratch.style        = sc.value("style",    "baby");
            p.structure.scratch.duration_bars = sc.value("duration_bars", 1);
            p.structure.scratch.stem         = sc.value("stem",     "mix");
        }

        // Parse build_up block
        if (s.contains("build_up")) {
            const auto& bu = s["build_up"];
            p.structure.build_up.enabled      = bu.value("enabled",      false);
            p.structure.build_up.length_bars  = bu.value("length_bars",  4);
            p.structure.build_up.add_riser    = bu.value("add_riser",    true);
            p.structure.build_up.filter_sweep = bu.value("filter_sweep", true);
        }
    }
    ```

    Use `j.value(key, default)` pattern throughout — same permissive parsing
