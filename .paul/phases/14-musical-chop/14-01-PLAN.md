```gherkin

When detect_beats(audio, sr) is called
```

## AC-2: Onset detection respects min_gap
```gherkin
Given a percussive audio array and sr
```

## AC-3: Bar detection aligns to beat grid
```gherkin
Given audio and beats_per_bar=4
When detect_bars(audio, sr, beats_per_bar=4) is called

```

## AC-4: Energy gate returns sample-aligned boolean mask
```gherkin
Given audio with silent and loud sections and threshold_db=-20
When detect_energy_gates(audio, sr, threshold_db=-20, hold_ms=50) is called
Then it returns a 1-D bool array of length == audio.shape[-1]
  And silent regions are False, loud regions are True
```

## AC-5: Structural segmentation returns non-overlapping ordered segments
```gherkin
Given audio of length ≥ 10s and n_segments=4
When detect_structure(audio, sr, n_segments=4) is called
Then it returns a list of (start_sec, end_sec) tuples
  And segments cover [0, duration) without gaps or overlaps
  And len(result) == n_segments (or fewer if audio too short)
```

<task type="auto">
  <name>Task 1: Create analysis.py with 5 detection functions</name>
  <files>python/server/remix/analysis.py</files>
  <action>
    Create the file with these 5 public functions — all accept (audio: np.ndarray, sr: int, **kwargs) patterns:

    **detect_beats(audio, sr) -> np.ndarray**
    - Mix to mono if stereo (average channels)
    - Call librosa.beat.beat_track(y=mono, sr=sr) → (tempo, beat_frames)
    - Return librosa.frames_to_time(beat_frames, sr=sr) as sorted float array
    - If beat_frames empty, return np.array([0.0])

    **detect_onsets(audio, sr, min_gap_ms=80, threshold=0.3) -> np.ndarray**
    - Mix to mono
    - onset_frames = librosa.onset.onset_detect(y=mono, sr=sr, delta=threshold, units="frames")
    - Convert to times via librosa.frames_to_time
    - Filter: keep onset[i] only if onset[i] - onset[i-1] >= min_gap_ms/1000
    - Return filtered array; if empty return np.array([0.0])

    **detect_bars(audio, sr, beats_per_bar=4) -> np.ndarray**
    - Call detect_beats(audio, sr) to get beat times
    - Slice every beats_per_bar-th beat starting from index 0: beat_times[::beats_per_bar]
    - Return result

    **detect_energy_gates(audio, sr, threshold_db=-20.0, hold_ms=50.0) -> np.ndarray**
    - Mix to mono
    - Compute frame-level RMS: librosa.feature.rms(y=mono) → shape (1, n_frames)
    - Convert RMS to dB: 20 * np.log10(rms + 1e-9)
    - Create frame-level bool mask: rms_db >= threshold_db
    - Apply hold: for each True frame, mark next hold_frames as True
      hold_frames = int(hold_ms / 1000 * sr / 512)  # default hop=512
    - Upsample bool mask from frame resolution to sample resolution via np.repeat
      samples_per_frame = 512
      mask_samples = np.repeat(frame_mask, samples_per_frame)
      Trim or pad to audio.shape[-1]
    - Return 1-D bool array of length audio.shape[-1]

    **detect_structure(audio, sr, n_segments=8) -> list[tuple[float, float]]**
    - Mix to mono
    - Compute MFCC feature matrix: librosa.feature.mfcc(y=mono, sr=sr, n_mfcc=13)
    - Use librosa.segment.agglomerative(mfcc, n_segments) → boundary_frames
    - Add frame 0 and last frame to boundaries
    - Convert to times; build (start, end) tuple list from adjacent boundaries
    - If fewer than n_segments possible, return what's available
    - Return list[tuple[float, float]]

    Avoid: calling librosa on empty arrays (guard with len check); importing pedalboard here (analysis.py is analysis-only, no DSP mutations).
