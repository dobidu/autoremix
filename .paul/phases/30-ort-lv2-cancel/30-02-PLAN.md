<objective>
## Goal
Add LV2 plugin format to the build and CI, producing `AutoRemix.lv2` bundles in
Linux (and macOS) release artifacts.

## Purpose
Linux DAW users (Ardour, Bitwig Studio, Carla, Zrythm) primarily use LV2. VST3 Linux
support in Ardour requires additional setup; LV2 is native. JUCE 7 supports LV2 via a
one-line FORMATS change. This expands AutoRemix's reach to Linux studio workflows
without any new code.

## Output
- `plugin/CMakeLists.txt`: `LV2` added to `FORMATS`
- `.github/workflows/release.yml`: LV2 bundle included in Linux (CPU + GPU) and macOS
  (CPU + GPU) package steps; `AutoRemix.lv2` in release artifact ZIPs
</objective>

<context>
@plugin/CMakeLists.txt
@.github/workflows/release.yml
</context>
