# Phase 20-01 — Audio Playback

## Objective

Add in-plugin audio playback for stems and final output, removing all reliance on the OS media player.

## Acceptance Criteria

- [ ] Per-stem play/stop toggle buttons in ScreenStemsReady (4 stems, mix simultaneously)
- [ ] ▶ Original / ▶ Remix buttons in ScreenRender Done state
- [ ] Button text updates to ■ while playing, reverts to ▶ when playback stops naturally (EOF)
- [ ] Exiting a screen stops all active playback
- [ ] No OS player launched anywhere

## Tasks

### T1 — Processor multi-source playback
- Add `MixerAudioSource` mixing 4 `StemPlayer` transports + 1 preview transport
- Implement `playStem(idx, file)`, `stopStem(idx)`, `isStemPlaying(idx)`, `stopAllStems()`
- `loadPreviewFile` / `togglePreview` / `stopPreview` / `isPreviewPlaying` for exclusive preview

### T2 — ScreenContext audio callbacks
- Add 7 callbacks: `play_preview`, `stop_preview`, `is_preview_playing`, `play_stem`, `stop_stem`, `is_stem_playing`, `stop_all_stems`
- Editor constructor wires all 7 to processor methods

### T3 — ScreenStemsReady play buttons
- Add `play_btn` (TextButton) to each stem row
- `RIGHT_COL_W` 280→320 to fit 3 toggle buttons
- `public juce::Timer` at 100ms polls `is_stem_playing(i)`, updates ▶/■ text
- `onEnter()` starts timer; `onExit()` stops timer + `stop_all_stems()`
- `handlePlay(idx)`: toggle stop/play per stem

### T4 — ScreenRender dual play buttons
- Replace single `preview_btn_` with `original_btn_` + `remix_btn_`
- `PreviewMode` enum: None / Original / Remix
- Timer multiplexed: 1000ms during Rendering (elapsed counter), 100ms during Done (poll `is_preview_playing`)
- `handleOriginalPlay()` / `handleRemixPlay()`: exclusive toggle
- `onExit()` stops preview
