## AC-3: EffectChainEngine walks chain + dispatches per stems

```gherkin
Given a separators::NativeStems with 4 buffers + a chain of EffectSteps
When process_effect_chain runs:
Then for each step:
  - if step.stems == ["all"]: sum the 4 stems, apply op once,
    divide by 4, redistribute to each stem
  - if step.stems == ["vocals"] etc.: apply op to that one stem
  - if step.stems is multiple names: apply op to each in turn
  And after all steps complete, do a weighted final mix using StemMix
  And output buffer is the weighted sum / 4
  And output is returned as a juce::AudioBuffer<float>
```

      static inline juce::AudioBuffer<float>
      chop_at_boundaries(const juce::AudioBuffer<float>& in,
                         std::vector<int> boundaries, int repeat)
      {
          // Port of _chop_at_boundaries from ops.py.
          // 1. Insert 0 + N at ends; sort & dedup; clamp to [0, N].
          // 2. For each segment [edges[i], edges[i+1]):
          //      copy that chunk `repeat` times (min 1) into a growing list.
          // 3. Concatenate. If total > N, trim. If total < N, zero-pad to N.
          // 4. Return same-shape buffer.
      }

    Public:

      inline juce::AudioBuffer<float>
      chop_beats(const juce::AudioBuffer<float>& in, double sr, const OpParams& p)
      {
          auto beats = analysis::detect_beats(in, sr);
          if (p.division != 1.0 && p.division > 0.0) {
              // Subdivide intervals into `steps = max(1, round(1/division))` parts
              const int steps = std::max(1, (int) std::round(1.0 / p.division));
              std::vector<double> extra;
              for (size_t i = 0; i + 1 < beats.size(); ++i) {
                  const double interval = beats[i + 1] - beats[i];
                  for (int s = 1; s < steps; ++s)
                      extra.push_back(beats[i] + s * interval / steps);
              }
              beats.insert(beats.end(), extra.begin(), extra.end());
              std::sort(beats.begin(), beats.end());
          }
          if ((size_t) p.offset_beats < beats.size())
              beats.erase(beats.begin(), beats.begin() + p.offset_beats);
          std::vector<int> boundaries;
          boundaries.reserve(beats.size());
          for (double t : beats)
              boundaries.push_back(std::clamp((int) (t * sr), 0, in.getNumSamples()));
          return chop_at_boundaries(in, std::move(boundaries), p.repeat);
      }

      inline juce::AudioBuffer<float>
      chop_onsets(const juce::AudioBuffer<float>& in, double sr, const OpParams& p)
      {
          auto onsets = analysis::detect_onsets(in, sr, p.min_gap_ms, p.threshold);
          std::vector<int> boundaries;
          boundaries.reserve(onsets.size());
          for (double t : onsets)
              boundaries.push_back(std::clamp((int) (t * sr), 0, in.getNumSamples()));
          return chop_at_boundaries(in, std::move(boundaries), p.repeat);
      }

      inline juce::AudioBuffer<float>
      chop_bars(const juce::AudioBuffer<float>& in, double sr, const OpParams& p)
      {
          auto bars = analysis::detect_bars(in, sr, p.beats_per_bar);
          std::vector<int> boundaries;
          boundaries.reserve(bars.size());
          for (double t : bars)
              boundaries.push_back(std::clamp((int) (t * sr), 0, in.getNumSamples()));
          return chop_at_boundaries(in, std::move(boundaries), p.repeat);
      }

      inline juce::AudioBuffer<float>
      gate_energy(const juce::AudioBuffer<float>& in, double sr, const OpParams& p)
      {
          auto mask = analysis::detect_energy_gates(in, sr, p.threshold_db, p.hold_ms);
          juce::AudioBuffer<float> out; out.makeCopyOf(in);
          const int N = std::min((int) mask.size(), out.getNumSamples());
          for (int c = 0; c < out.getNumChannels(); ++c) {
              auto* d = out.getWritePointer(c);
              for (int i = 0; i < N; ++i)
                  if (!mask[(size_t) i]) d[i] = 0.0f;
          }
          return out;
      }

      inline juce::AudioBuffer<float>
      structural_cut(const juce::AudioBuffer<float>& in, double /*sr*/, const OpParams& p)
      {
          // Native fallback: uniform N segments instead of MFCC-clustered.
          // Reorder per mode, concat, return.
          const int N         = in.getNumSamples();
          const int ch        = in.getNumChannels();
          const int nSeg      = std::max(2, p.n_segments);
          const int segLen    = N / nSeg;
          if (segLen <= 0) { juce::AudioBuffer<float> out; out.makeCopyOf(in); return out; }

          // Build segment indices [start, end)
          std::vector<std::pair<int,int>> segs;
          segs.reserve((size_t) nSeg);
          for (int i = 0; i < nSeg; ++i) {
              const int s = i * segLen;
              const int e = (i == nSeg - 1) ? N : std::min(N, s + segLen);
              segs.emplace_back(s, e);
          }

          // Reorder per mode
          if (p.mode == "reverse") {
              std::reverse(segs.begin(), segs.end());
          } else if (p.mode == "shuffle") {
              std::mt19937 rng(42);
              std::shuffle(segs.begin(), segs.end(), rng);
          } else if (p.mode == "repeat_first" && segs.size() > 1) {
              // [s0, s1, s0, s2, s0, s3, ...]
              std::vector<std::pair<int,int>> woven;
              woven.reserve(segs.size() * 2);
              woven.push_back(segs[0]);
              for (size_t i = 1; i < segs.size(); ++i) {
                  woven.push_back(segs[i]);
                  woven.push_back(segs[0]);
              }
              segs = std::move(woven);
          }

          // Concatenate
          int total = 0;
          for (auto& [s, e] : segs) total += e - s;
          juce::AudioBuffer<float> out(ch, total);
          int pos = 0;
          for (auto& [s, e] : segs) {
              const int n = e - s;
              for (int c = 0; c < ch; ++c)
                  out.copyFrom(c, pos, in, c, s, n);
              pos += n;
          }
          return out;
      }

## Open Questions (revisit during APPLY)

- **detail:: visibility**: 24-01 put apply_iir_filter / apply_reverb in
  `engines::detail`. NativeEffectOps will reach into that. Either expose
  them at `engines::` scope or duplicate as tiny wrappers in
  NativeEffectOps.h. Default: reference `engines::detail::` directly
  (legal since detail is just a named nested namespace, not anonymous).
- **structural_cut quality**: uniform N-segment fallback is the v1
  shortcut. If Phase 27 A/B tests reveal that structural_loop sounds
  significantly worse than v3, port detect_structure (MFCC +
  agglomerative clustering) and replace the fallback in this op.
- **OpParams approach**: single struct with all fields vs std::variant
  + per-op typed args. Single struct is simpler and matches v3's dict
  semantics (extra fields ignored). std::variant would be type-safe but
  10× the code.
