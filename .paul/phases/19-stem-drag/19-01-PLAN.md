# Goal
Add drag-to-DAW on stem rows in ScreenStemsReady. Clicking and dragging the waveform/name area of any stem row initiates an OS-level file drag of the corresponding temp WAV, so users can drop it directly onto a DAW track.

# Tasks

## T1 — Add mouse tracking members to ScreenStemsReady
- `int drag_stem_idx_ = -1;` — row being dragged (-1 = none)
- `juce::Point<int> mouse_down_pos_;`

## T2 — Override mouseDown
- If x is in right col controls region: set drag_stem_idx_ = -1, return
- Else: set drag_stem_idx_ = stemRowAtY(e.y), record mouse_down_pos_

## T3 — Override mouseDrag
- Guard: drag_stem_idx_ < 0, return
- Guard: getDistanceFromDragStart() < 8px threshold, return
- Resolve path via stemPath(drag_stem_idx_)
- Guard: path empty or file doesn't exist, return
- Reset drag_stem_idx_ = -1 (prevent re-trigger mid-drag)
- Call juce::DragAndDropContainer::performExternalDragDropOfFiles({path}, false)

## T4 — Add private helpers
```cpp
int stemRowAtY(int y) const;       // maps y → row index [0-3] or -1
juce::String stemPath(int idx) const; // maps index → filesystem path string
```

## T5 — Build clean, verify no regressions
