## Goal
Add three new built-in remix styles as effect-chain presets: Phonk, Jersey Club,
and Nightcore. All use the existing effect-chain DSL — no new Python code.

## Purpose
v8 milestone "More remix styles" gives users more creative options beyond the three
legacy engines (Chopped & Screwed, Slowed + Reverb, Drum & Bass).

## Output
Three new JSON files in `python/server/presets/`:
- `phonk.json` — Memphis phonk: slowed, dark, bass-heavy, woozy reverb, vocal stutter
- `jersey_club.json` — Jersey Club: sped up, tight onset vocal chop, 808 bass forward
- `nightcore.json` — Nightcore: fast + bright, pitched-up vocals
Sidecar serves all six presets after restart. UI picks them up automatically.

  <action>
    Create `python/server/presets/phonk.json`:

    ```json
    {
      "id": "phonk",
      "version": "1.0",
      "name": "Phonk",
      "description": "Memphis phonk: slowed and dark with woozy bass, dusty reverb, and half-beat vocal stutter.",
      "author": "AutoRemix",
      "tags": ["phonk", "memphis", "trap", "dark", "slow"],
      "engine": "effect_chain",
      "params": {
        "tempo_factor": 0.88,
        "pitch_shift_semi": -3.0,
        "reverb_mix": 0.25,
        "chop_interval_ms": 0.0,
        "bass_boost_db": 0.0,
        "drums_tempo_factor": 1.0
      },
      "stem_mix": { "vocals": 1.0, "drums": 1.1, "bass": 1.2, "other": 0.7 },
      "effects": [
        { "op": "time_stretch", "stems": ["vocals","drums","bass","other"], "params": { "factor": 0.88 } },
        { "op": "pitch_shift",  "stems": ["vocals","bass","other"],         "params": { "semitones": -3.0 } },
        { "op": "bass_boost",   "stems": "bass",                           "params": { "db": 8.0 } },
        { "op": "bass_boost",   "stems": "drums",                          "params": { "db": 4.0 } },
        { "op": "reverb",       "stems": "vocals",                         "params": { "mix": 0.25, "room_size": 0.85 } },
        { "op": "chop_beats",   "stems": "vocals",                         "params": { "division": 0.5, "repeat": 2, "offset_beats": 0 } }
      ]
    }
    ```

    Design rationale:
    - 0.88x tempo ≈ authentic Memphis phonk speed (85-90 BPM from 95-100 source)
    - -3 semitones on vocals + bass + other → woozy, dark texture; drums kept dry/punchy
    - bass_boost +8 on bass, +4 on drums → heavy 808-like low end
    - reverb mix 0.25, room_size 0.85 → dusty, cavernous but not washy
    - half-beat vocal chop (division 0.5) → signature phonk stutter every half beat
  </action>

    ```json
      "effects": [
        { "op": "time_stretch", "stems": ["drums","bass","other"], "params": { "factor": 1.25 } },
        { "op": "chop_onsets",  "stems": "vocals",                "params": { "min_gap_ms": 50, "threshold": 0.20, "repeat": 3 } },
        { "op": "bass_boost",   "stems": "bass",                  "params": { "db": 6.0 } },
        { "op": "bass_boost",   "stems": "drums",                 "params": { "db": 3.0 } }
      ]
    }
    ```

    Design rationale:
    - 1.25x on drums/bass/other → pushes to ~125 BPM from 100 BPM source (Jersey is 130-155 BPM)
    - Vocals NOT time-stretched → detached, floating effect over fast rhythm section
    - onset chop on vocals with tight min_gap (50ms), low threshold (0.20), repeat=3 → rapid
      triplet-style stutter that defines the Jersey Club sound
    - bass_boost +6 on bass, +3 on drums → 808 heavy mix
    - stem_mix: drums 1.3 forward, other 0.6 back → punchy, minimal atmosphere
  </action>
